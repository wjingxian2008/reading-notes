**使用gradle快速搭建项目**
  1. gradle init --type java-library
  2. 在gradle.build 的plugins 中添加配置，id 'idea'
   
   ``
     plugins {
       // Apply the java-library plugin to add support for Java Library
       id 'java-library'
       id 'idea'
     }
   ``

  3. open *.ipr
